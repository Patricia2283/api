# api
API Rest desenvolvida  nas aulas de Spring Boot 3 da Alura, para uma clínica médica fictícia.
Foi utlilizado o CRUD: cadastro, listagem, atualização e exclusão de informações de médicos e pacientes.

<br>°Java 17<br>
<br>°Lombok<br>
<br>°MySQL/ Flyway<br>
<br>°JPA/Hibernate<br>
<br>°Maven<br>
<br>°Insomnia
